## Quick Example

```jsx
import { useControls } from 'leva'

function MyComponent() {
  const { name, age, color } = useControls({
    name: 'World',
    age: 25,
    color: '#ff0000',
  })

  return (
    <div style={{ color }}>
      Hello {name}, you are {age} years old!
    </div>
  )
}
```

## Why Leva?

Leva makes it incredibly easy to add a GUI to your React applications. Whether you're prototyping, debugging, or building creative tools, Leva provides an intuitive interface that adapts to your needs.

- **Smart Type Recognition**: Just pass your values, Leva figures out the best input type
- **Extensible**: Create custom plugins for specialized inputs
- **Customizable**: Theme the entire GUI to match your application
- **No Boilerplate**: Start using it immediately without configuration
